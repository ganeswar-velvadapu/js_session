Variables in JavaScript

Variables are containers for storing data values.

Three ways to declare variables:

// var - function scoped (older, avoid when possible)
var oldVariable = "I'm old school";

// let - block scoped (use for values that will change)
let counter = 0;

// const - block scoped (use for values that won't change)
const API_KEY = "abc123";

Variable Scopes & Hoisting

var

  • Function-scoped
  • Hoisted with undefined value
  • Can be redeclared

let & const

  • Block-scoped
  • Hoisted but not initialized (TDZ)
  • Cannot be redeclared in same scope
  • const must be initialized and can't be reassigned

Best Practice: Use const by default, let when needed, avoid var

JavaScript Data Types

JavaScript has 8 data types:

Primitive Types

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Symbol
  • BigInt

Reference Type

  • Object
    • Object literals
    • Arrays
    • Functions
    • Dates
    • RegExp
    • Map, Set, etc.

Primitive Types Examples

// String - text data
const name = "JavaScript";

// Number - integers and floating points
const integer = 42;
const float = 3.14;

// Boolean - true/false
const isActive = true;

// Undefined - variable declared but not assigned
let future;

// Null - intentional absence of value
const empty = null;

// Symbol - unique identifier
const uniqueId = Symbol("id");

// BigInt - for large integers
const bigNumber = 9007199254740991n;

Reference Types Example

// Object - collection of key-value pairs
const person = {
  name: "John",
  age: 30
};

// Array - ordered collection of items
const fruits = ["apple", "orange", "banana"];

// Function - callable object
function greet() {
  return "Hello!";
}

// Date object
const today = new Date();

Checking Types

// Using typeof operator
console.log(typeof "Hello");     // "string"
console.log(typeof 42);         // "number"
console.log(typeof true);       // "boolean"
console.log(typeof undefined);  // "undefined"
console.log(typeof null);       // "object" (⚠️ historical bug)
console.log(typeof Symbol());   // "symbol"
console.log(typeof {});         // "object"
console.log(typeof []);         // "object" (arrays are objects)
console.log(typeof function(){}); // "function"

// For arrays, use Array.isArray()
console.log(Array.isArray([])); // true

Type Conversion

Converting values from one data type to another.

Explicit Conversion

When you manually convert types:

  • String()
  • Number()
  • Boolean()
  • parseInt()/parseFloat()

Implicit Conversion

When JavaScript automatically converts types:

  • During operations (+, -, *, /)
  • During comparisons (==, !=)
  • In conditional statements

String Conversion

// To String
String(123);          // "123"
String(true);         // "true"
String(null);         // "null"
String(undefined);    // "undefined"
String([1, 2, 3]);    // "1,2,3"

// Alternative methods
123.toString();       // "123"
(123).toString();     // "123"
[1, 2, 3].toString(); // "1,2,3"

// Template literals
`The value is ${123}`; // "The value is 123"

Numeric Conversion

// To Number
Number("123");        // 123
Number("12.3");       // 12.3
Number("");           // 0
Number(true);         // 1
Number(false);        // 0
Number(null);         // 0
Number(undefined);    // NaN
Number("text");       // NaN
Number([1]);          // 1
Number([1, 2]);       // NaN

// Parsing functions
parseInt("123px");    // 123
parseFloat("12.3em"); // 12.3

// Unary plus operator
+"123";               // 123

Boolean Conversion

// To Boolean
Boolean(1);           // true
Boolean(0);           // false
Boolean("hello");     // true
Boolean("");          // false
Boolean(null);        // false
Boolean(undefined);   // false
Boolean(NaN);         // false
Boolean([]);          // true (empty array is truthy)
Boolean({});          // true (empty object is truthy)

// Falsy values: false, 0, "", null, undefined, NaN
// Everything else is truthy

// Double negation shorthand
!!1;                  // true
!!0;                  // false

Implicit Type Conversion

// String concatenation with +
"5" + 2;              // "52" (number converts to string)
2 + "5";              // "25" (number converts to string)

// Numeric operations (except +)
"5" - 2;              // 3 (string converts to number)
"5" * 2;              // 10 (string converts to number)
"5" / 2;              // 2.5 (string converts to number)

// Comparisons
"5" == 5;             // true (loose equality)
"5" === 5;            // false (strict equality - no conversion)

// Boolean context (if, while, ternary)
if ("hello") {        // converts to true
  // will execute
}

JavaScript Operators

Symbols that perform operations on operands (values and variables).

Operator Type Examples
Assignment =, +=, -=, *=, /=, %=
Arithmetic +, -, *, /, %, **, ++, --
Comparison ==, ===, !=, !==, >, <,>=, <=< /td>
Logical &&, ||, !
Bitwise &, |, ~, ^, <<,>>, >>>
Special ?. (optional chaining), ?? (nullish coalescing)

Arithmetic Operators

// Basic arithmetic
let a = 10;
let b = 3;

a + b;    // 13 (addition)
a - b;    // 7 (subtraction)
a * b;    // 30 (multiplication)
a / b;    // 3.3333... (division)
a % b;    // 1 (modulus - remainder)
a ** b;   // 1000 (exponentiation - 10^3)

// Increment/decrement
let counter = 5;
counter++;  // 5 (post-increment, returns then adds 1)
++counter;  // 7 (pre-increment, adds 1 then returns)
counter--;  // 7 (post-decrement, returns then subtracts 1)
--counter;  // 5 (pre-decrement, subtracts 1 then returns)

Comparison Operators

// Loose equality (with type conversion)
5 == "5";      // true
0 == false;    // true
"" == 0;       // true
null == undefined; // true

// Strict equality (no type conversion)
5 === "5";     // false
0 === false;   // false

// Inequality
5 != "6";      // true
5 !== 5;       // false

// Relational
5 > 3;         // true
5 < 3;         // false
5 >= 5;        // true
5 <= 6;        // true

Best Practice: Always use strict equality (===, !==) to avoid unexpected type conversions

Logical Operators

// Logical AND (&&) - returns first falsy value or last value
true && true;    // true
true && false;   // false
"Hello" && 42;   // 42 (both truthy, returns last value)
0 && "Hello";    // 0 (first value is falsy, returns it)

// Logical OR (||) - returns first truthy value or last value
true || false;   // true
false || false;  // false
"Hello" || 42;   // "Hello" (first value is truthy, returns it)
0 || "";         // "" (both falsy, returns last value)

// Logical NOT (!) - converts to boolean and inverts
!true;           // false
!0;              // true
!!"Hello";       // true (double negation = boolean conversion)

Modern JavaScript Operators

// Nullish coalescing operator (??) - returns right side only when left is null/undefined
let username = null ?? "Guest";          // "Guest"
let count = 0 ?? 42;                     // 0 (0 is not nullish)
let empty = "" ?? "Default";             // "" (empty string is not nullish)

// Optional chaining (?.) - safely access nested properties
const user = { 
  profile: { 
    name: "John" 
  } 
};

user.profile.name;                      // "John"
user.settings?.theme;                   // undefined (no error)
user.getFullName?.();                   // undefined (no error)

// Without optional chaining would cause error:
// user.settings.theme // TypeError: Cannot read property 'theme' of undefined

Math Object

Built-in object for mathematical constants and functions.

Common Constants

  • Math.PI (π ≈ 3.14159...)
  • Math.E (e ≈ 2.71828...)
  • Math.SQRT2 (√2 ≈ 1.414...)
  • Math.LN2 (ln(2) ≈ 0.693...)
  • Math.LN10 (ln(10) ≈ 2.302...)

Common Methods

  • Math.round(), floor(), ceil()
  • Math.min(), max()
  • Math.abs(), pow(), sqrt()
  • Math.random()
  • Math.sin(), cos(), tan()

Math Object Examples

// Rounding methods
Math.round(4.7);     // 5
Math.round(4.4);     // 4
Math.floor(4.7);     // 4 (always rounds down)
Math.ceil(4.3);      // 5 (always rounds up)
Math.trunc(4.7);     // 4 (removes decimal part)

// Min and Max
Math.min(2, 5, 1, 8, 3);    // 1
Math.max(2, 5, 1, 8, 3);    // 8

// Power and Square Root
Math.pow(2, 3);     // 8 (2³)
Math.sqrt(9);       // 3 (√9)
Math.cbrt(27);      // 3 (∛27)

// Random (0 to less than 1)
Math.random();      // 0.123456789... (random number)

Working with Math.random()

// Random number between 0 and 1 (not including 1)
Math.random();                            // 0.123456789...

// Random integer from 0 to max (not including max)
function getRandomInt(max) {
  return Math.floor(Math.random() * max);
}
getRandomInt(10);                        // 0-9

// Random integer from min to max (inclusive)
function getRandomIntInclusive(min, max) {
  min = Math.ceil(min);
  max = Math.floor(max);
  return Math.floor(Math.random() * (max - min + 1) + min);
}
getRandomIntInclusive(1, 10);           // 1-10

// Random element from array
function getRandomElement(array) {
  return array[Math.floor(Math.random() * array.length)];
}
getRandomElement(["red", "green", "blue"]);  // random color

Date Object

Built-in object for working with dates and times.

Creating Date Objects

// Current date and time
const now = new Date();                  // current date & time

// Specific date (year, month [0-11], day, hour, minute, second, ms)
const birthday = new Date(1990, 0, 15);  // Jan 15, 1990

// From timestamp (milliseconds since Jan 1, 1970)
const timestamp = new Date(1609459200000); // Dec 31, 2020

// From date string
const christmas = new Date("2023-12-25"); // Christmas 2023
const halloween = new Date("October 31, 2023"); // Halloween 2023

Date Methods: Getters

const date = new Date("2023-05-15T14:30:45");

// Date components (local time)
date.getFullYear();    // 2023
date.getMonth();       // 4 (0-based, so 4 = May)
date.getDate();        // 15
date.getDay();         // 1 (0 = Sunday, 1 = Monday, etc.)

// Time components (local time)
date.getHours();       // 14 (2 PM)
date.getMinutes();     // 30
date.getSeconds();     // 45
date.getMilliseconds(); // 0

// Timestamp
date.getTime();        // milliseconds since Jan 1, 1970

// UTC/GMT methods
date.getUTCHours();    // UTC hours (may differ from local)

Date Methods: Setters

const date = new Date();

// Set date components
date.setFullYear(2025);
date.setMonth(11);     // December (0-based)
date.setDate(31);

// Set time components
date.setHours(23);
date.setMinutes(59);
date.setSeconds(59);

// Set by timestamp
date.setTime(1735689599000);  // Dec 31, 2024 23:59:59

// UTC variants are also available
date.setUTCHours(0);

Date Formatting & Operations

const date = new Date();

// Common formatting methods
date.toString();           // "Mon Apr 06 2025 12:34:56 GMT+0000"
date.toDateString();       // "Mon Apr 06 2025"
date.toTimeString();       // "12:34:56 GMT+0000"
date.toISOString();        // "2025-04-06T12:34:56.789Z"
date.toLocaleString();     // "4/6/2025, 12:34:56 PM"
date.toLocaleDateString(); // "4/6/2025"
date.toLocaleTimeString(); // "12:34:56 PM"

// Date arithmetic (in milliseconds)
const tomorrow = new Date();
tomorrow.setDate(tomorrow.getDate() + 1);

// Calculate difference between dates (in milliseconds)
const diff = tomorrow - date;  // 86400000 (1 day in ms)

Arrays

Ordered collections of values accessed by index (starting at 0).

Creating Arrays

// Array literal (preferred)
const fruits = ["apple", "banana", "orange"];

// Array constructor (less common)
const numbers = new Array(1, 2, 3, 4, 5);

// Empty array
const empty = [];

// Array with preset length but no values
const preset = new Array(5);  // [empty × 5]

// Array from other iterable
const chars = Array.from("hello");  // ["h", "e", "l", "l", "o"]
const items = Array.of(1, 2, 3);    // [1, 2, 3]

Accessing Array Elements

const fruits = ["apple", "banana", "orange", "mango"];

// Access by index
fruits[0];                // "apple"
fruits[2];                // "orange"
fruits[fruits.length - 1]; // "mango" (last element)

// Out of bounds
fruits[10];               // undefined (no error)

// Nested arrays
const matrix = [[1, 2], [3, 4]];
matrix[0][1];             // 2

// Modifying elements
fruits[1] = "pear";       // ["apple", "pear", "orange", "mango"]

// Length property
fruits.length;            // 4

// Extend array by setting length
fruits.length = 6;        // ["apple", "pear", "orange", "mango", empty × 2]

// Truncate array by setting length
fruits.length = 2;        // ["apple", "pear"]

Basic Array Methods

const fruits = ["apple", "banana", "orange"];

// Adding elements
fruits.push("mango");         // 4 (returns new length)
                              // ["apple", "banana", "orange", "mango"]
fruits.unshift("strawberry"); // 5 (returns new length)
                              // ["strawberry", "apple", "banana", "orange", "mango"]

// Removing elements
fruits.pop();                 // "mango" (returns removed element)
                              // ["strawberry", "apple", "banana", "orange"]
fruits.shift();               // "strawberry" (returns removed element)
                              // ["apple", "banana", "orange"]

// Finding elements
fruits.indexOf("banana");     // 1
fruits.includes("orange");    // true

// Combining arrays
const more = ["grape", "kiwi"];
const combined = fruits.concat(more);  // ["apple", "banana", "orange", "grape", "kiwi"]

// Extracting parts (slice does not modify original)
fruits.slice(1, 3);           // ["banana", "orange"]

Array Methods for Transformation

const numbers = [1, 2, 3, 4, 5];

// map - transform each element (returns new array)
const doubled = numbers.map(num => num * 2);
// [2, 4, 6, 8, 10]

// filter - keep elements that pass a test (returns new array)
const evens = numbers.filter(num => num % 2 === 0);
// [2, 4]

// reduce - accumulate values (returns single value)
const sum = numbers.reduce((acc, num) => acc + num, 0);
// 15

// some - tests if any element passes (returns boolean)
const hasEven = numbers.some(num => num % 2 === 0);
// true

// every - tests if all elements pass (returns boolean)
const allPositive = numbers.every(num => num > 0);
// true
